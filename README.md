University Listing Website

Overview

This is a MERN stack project for listing universities and their respective departments. The project includes a user-friendly interface for browsing universities and an admin panel for managing universities and departments.

Features

University listing with details

Department listing under each university

University Admin Panel:

CRUD operations for universities

Image uploads for universities

Department Admin Panel:

CRUD operations for departments

Bootstrap-based UI for consistency with the University Admin Panel

Tech Stack:

Frontend: React (with Vite), Bootstrap (for admin panel UI)

Backend: Node.js, Express.js, MongoDB (Mongoose ORM)

API Handling: Axios

Notifications: SweetAlert2
